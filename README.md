#Hashbot
#####Hashbot for LBRYs slack, sends out messages about the current mining process every hour or
#####when the command !hash gets detected.

######Installation
**Requirements:**
**node and npm > 0.12.x**
```
npm install
Edit the bot.js and add api key.
node bot.js
```
